## Dependencies

- git
- firefox

## Installation

Navigate into your firefox profile directory
```bash
git init
git remote add origin https://github.com/ghoulboii/firefox
git fetch
git checkout origin/master -ft
./updater.sh  # If using Linux
./updater.bat #If using Windows
```
